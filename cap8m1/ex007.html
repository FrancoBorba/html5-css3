Principais formatações

Negrito/ destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semantico)

Nesta frase temos um Termo em destaque usando a tag <strong> (semantico)

Italico/ enfase

Nesta frase temos um termo em italico , usando a tag <I>(não semantica)

Nesta frase temos um termo em enfasê usando a tag <EM> (semantico)

Texto marcado

Podemos também criar um texto marcado utilizando a tag <MARK >

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste paragrafo
A tag <big> caiu em desuso porém a tag <small> continua-se usando

Texto Excluido

Podemos marcar um texto como exluido utilizando a tag <del> , para demonstrar que ele deve ser lido mas não considerado

Texto inserido

Podemos marcar um texto como inserido utilizando a tag <ins> para dar uma ênfase ou demonstar que ele foi adicionado depois

Texto sobrescrito

Para inserir coisas como x20 + 3

Texto subscrito

Para inserir coisas como h20
